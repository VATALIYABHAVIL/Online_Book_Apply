Online Book Apply System 📖🖋️
Online Book Apply is an efficient online application system developed to facilitate authors to submit their manuscripts or anyone to apply for book-related resources. 
It is an online application system that helps to ease the application process by providing a safe and efficient platform to applicants and administrators.


✨ Features
Digital Submission Form: It is an online form where an author can submit his/her book title, genre, summary, and manuscript file.
Real-time Status Updates: It allows applicants to view the status of their application (e.g., Submitted, Under Review, Accepted).
Admin Dashboard: It is an online platform provided to publishers/librarians to manage all applications submitted to them.
File Upload: It is an online system to securely upload document formats such as PDF or DOCX files containing manuscripts.
Search & Filter: It is an online platform to easily manage all applications based on genre, author name, or date of submission.


🛠️ Tech Stack
Front-end: React/Next.js
Back-end: Node.js/Express
Database: SQL/Firebase
Storage: Firebase Storage


📋 How It Works
User Registration: The author has to create an account to begin with.
Submit Details: The author has to fill out the "Book Apply" form with all the publication details.
File Upload: The manuscript is uploaded directly to cloud storage.
Review Process: The administrators get a notification to review the submission from their own dashboard.


🛡️ Use Cases
Publishing Houses: Handling large numbers of manuscript submissions is made easy.
Libraries: New book requests or membership applications can be handled easily.
Academic Institutions: Grant applications or thesis submissions can be made hassle-free.
